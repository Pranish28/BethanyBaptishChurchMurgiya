# Bethany Baptist Church Murgiya Website

A professional website for Bethany Baptist Church Murgiya built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Professional styling with modern fonts and icons
- Sections for Home, About Us, and Our Services

## How to View

Simply open `index.html` in your web browser to view the website.

## Structure

- `index.html` - Main HTML file
- `styles.css` - CSS stylesheets
- `script.js` - JavaScript for interactivity

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts for typography

## Customization

To customize the content, edit the text in `index.html`. For styling changes, modify `styles.css`. For functionality changes, edit `script.js`.
